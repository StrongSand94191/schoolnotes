# Properties of Acids
- Acids are protons
- Acids have a pH lower than 7
- Acids taste sour
- Acids affect indicators
	- Blue litmus turns red
	- Methyl orange turns red
- Acids react with active metals, producing H2
- Acids react with carbonates
- Acids neutralize bases
